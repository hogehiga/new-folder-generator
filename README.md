#What is this?
Create different kinds of "new folder".

#How to use
- `./nfg.bash` Makes three new folders in current directory.

- `./nfg.bash -n 10` Makes ten new folders in current directory.

- `./nfg.bash -d /tmp` Makes new folders in /tmp.
